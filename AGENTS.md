# AGENTS - 工作规范

本文件定义了 FactorMiner 智能体的工作规范，所有分析和计算必须遵循这些规则。

---

## 1. 数据使用规范

### 1.1 复权价格规则

**核心规则：**

| 场景 | 使用复权类型 | 说明 |
|------|-------------|------|
| **因子计算/IC评估** | `hfq` (后复权) | 保持历史价格可比性，适合回测 |
| **结果展示** | 同时展示 `hfq` 和 除权价 | 给用户完整的参考信息 |

**原因：**
- 后复权 (`hfq`)：保持历史价位的真实感，避免除权缺口影响计算
- 除权价：反映实际股价水平，便于用户理解

**代码规范：**

```python
# ❌ 错误：计算时使用除权价
loader = AShareDataLoader(adj=None)
df = loader.load()

# ✅ 正确：计算时使用后复权，展示时同时提供除权价
loader_hfq = AShareDataLoader(adj="hfq")
df_hfq = loader_hfq.load()

loader_raw = AShareDataLoader(adj=None)
df_raw = loader_raw.load()

# 因子计算使用 hfq 数据
# 结果展示时对比 hfq 和 除权价
```

### 1.2 多因子分析规范

**分析清单：**

1. **趋势指标**
   - MA5, MA10, MA20, MA60 均线位置
   - 均线多头/空头排列判断

2. **动量指标**
   - RSI3, RSI5（3日/5日相对强弱）
   - 公式: `RSI = Mean(returns, N) / (Std(returns, N) + 0.001)`

3. **资金流向**
   - 吸筹强度 = `net_mf_vol / (volume + 1)`
   - 正值表示主力净流入

4. **支撑压力**
   - 20日/60日高低点
   - 止损止盈参考

### 1.3 IC 评估指标使用

| 指标 | 用途 |
|------|------|
| **IC Mean** | 判断因子方向（正/负） |
| **IC Abs Mean** | 判断因子强度（> 0.03 有效） |
| **ICIR** | 判断稳定性（> 0.5 稳定） |
| **Win Rate** | 判断胜率（> 55% 优秀） |

## 2. 股票筛选规范

### 2.1 市值 + RSI3 组合筛选

**综合评分公式：**

```
综合得分 = 0.4 × 小市值排名 + 0.6 × 高RSI3排名
```

**筛选流程：**

1. 计算 RSI3 = `Mean(returns, 3) / (Std(returns, 3) + 0.001)`
2. 按成交额估算市值（成交额越小 = 市值越小）
3. 综合排名取 TOP N

### 2.2 风险等级

| RSI3 范围 | 风险等级 | 建议 |
|-----------|----------|------|
| > 10 | 🔴 高 | 回避或轻仓 |
| 3 ~ 10 | 🟡 中 | 谨慎操作 |
| < 3 | 🟢 低 | 稳健布局 |

### 2.3 止盈止损策略

| 策略 | 计算方法 |
|------|----------|
| **止损价** | `现价 - 2 × 波动率` |
| **止盈价** | `max(现价 + 3×波动率, 20日高点 × 98%)` |
| **波动率** | `20日收益率标准差` |

## 4. 知识库维护

### 4.1 已验证有效的因子

| 因子 | IC Mean | Win Rate | 适用场景 |
|------|---------|----------|----------|
| RSI3 | +0.45 | 79.3% | 短期动量 |
| RSI5 | +0.36 | 75.5% | 中期动量 |

### 4.2 常见问题处理

| 问题 | 解决方案 |
|------|----------|
| NaN/NaT 错误 | 使用 `df.to_numpy(dtype=np.float64, na_value=0.0)` |
| DSL 找不到特征 | 检查 data_dict 是否加 `$` 前缀 |
| IC 全 NaN | 确保 M > 30（股票数量足够） |
