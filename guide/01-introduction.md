# 01 - FactorMiner 简介

## 什么是 FactorMiner？

**FactorMiner** 是一个基于大语言模型（LLM）的量化因子挖掘框架，能够从市场数据中自动发现具有可解释性的 Alpha 因子。

```
┌─────────────────────────────────────────────────────────────────┐
│                     FactorMiner 工作流程                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   市场数据 ──▶ 表达式树 ──▶ 因子评估 ──▶ 因子库                   │
│                    ▲         │                                 │
│                    │         ▼                                 │
│              ┌─────┴─────┐                                    │
│              │   LLM     │ ◀── 经验记忆                         │
│              │  生成器    │                                    │
│              └───────────┘                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 核心特性

| 特性 | 说明 |
|------|------|
| **LLM 驱动** | 使用大语言模型生成候选因子 |
| **经验记忆** | 从历史挖掘经验中学习 |

## 快速概念

### 什么是因子（Factor）？

因子是一个**数值公式**，用于预测股票未来收益。例如：

```
因子 = 收盘价 / 20日均线 - 1
公式 = $close / Mean($close, 20) - 1
```

这个因子衡量当前价格相对于近期均线的偏离程度。

### 表达式树

因子在 FactorMiner 中以**表达式树**存储：

```
表达式: $close / Mean($close, 20) - 1

        ┌──► $close (LeafNode)
        │
    div ◄── Mean ──┬──► $close (LeafNode)
        │              └──── 20 (ConstantNode)
        │
    neg
        │
    const(-1)
```

### IC (Information Coefficient)

IC 是衡量因子预测能力的指标：

```
IC = corr(因子值, 未来收益)
```

- IC > 0 表示正相关
- IC 绝对值越大，预测能力越强
- 通常认为 |IC| > 0.02 有意义
