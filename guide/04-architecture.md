## Agent 模块

负责 LLM 驱动的因子生成：

```
┌─────────────────┐
│  PromptBuilder │  构建提示（DSL语法 + 记忆）
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   LLM Provider  │  OpenAI / Anthropic / Mock
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ Output Parser    │  解析 LLM 输出
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ CandidateFactor │  候选因子
└─────────────────┘
```

## Memory 模块

经验记忆系统：

```
ExperienceMemory
       │
       ├── MemoryStore (存储)
       │      ├── SuccessPattern (成功模式)
       │      ├── ForbiddenDirection (禁止方向)
       │      └── StrategicInsight (战略洞察)
       │
       ├── Retrieval (检索)
       │      ├── 基础检索 (Paper Lane)
       │      └── KG + Embeddings (Helix Lane)
       │
       ├── Formation (形成)
       │      └── 从评估轨迹提取新记忆
       │
       └── Evolution (演化)
              └── 根据反馈调整记忆
```

## Evaluation 模块

多阶段评估管道：

```
Stage 1 ──▶ Stage 2 ──▶ Stage 2.5 ──▶ Stage 3 ──▶ Stage 4
  │          │           │            │          │
  ▼          ▼           ▼            ▼          ▼
Fast IC   Correlation  Replacement   Dedup     Full IC
筛选      检查        检查        去重      验证
```

## 双车道对比

| 组件 | Paper Lane | Helix Lane |
|------|-----------|------------|
| 循环 | RalphLoop | HelixLoop |
| 记忆 | 基础检索 | + KG + Embeddings |
| 评估 | IC/ICIR | + Causal + Regime |
| 生成 | 基础 | + 辩论 |
| 规范化 | 无 | SymPy |

## 配置优先级

```
defaults.yaml → user_config.yaml → CLI overrides
    (最低)           (中)           (最高)
```
